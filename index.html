Right lads. I've been building something absolutely mental for the last two weeks and I need you all to have it immediately.

It's called OpenClaw. It's an open-source AI assistant that runs on a server, connects to your Telegram (or Discord/WhatsApp/whatever), and basically becomes your second brain. It reads your emails, manages your calendar, remembers everything, builds software, drafts replies, and runs overnight while you sleep.

I'm not exaggerating. I have not hired a single person. I'm operating like a 15-person team with one AI assistant I talk to on Telegram.

It took me about 20 minutes to set up. Here's how you do it.

STEP 1

Get an Anthropic API Key 2 min

This is the brain. Claude is the AI model that powers everything. You need an API key.

  • Go to console.anthropic.com
  • Sign up / sign in
  • Go to API Keys ‚Üí Create a new key
  • Copy it somewhere safe. You'll need it in Step 3.
  • Add ¬£20 credit to start (this lasts ages - I spend about ¬£5-10/day and I hammer it)
üí° The model you want is Claude Sonnet 4 to start. It's the sweet spot of smart + cheap. You can upgrade to Opus later if you want the big brain.
STEP 2

Create a Telegram Bot 2 min

This is how you'll talk to your AI. Through Telegram. Like texting a mate.

  • Open Telegram and search for @BotFather
  • Send /newbot
  • Give it a name (e.g., "Atlas" or "Jarvis" or "Kevin" - whatever you want)
  • Copy the bot token it gives you (looks like 123456789:AAHdqTcvCH1vGWJx...)
üí° This is YOUR bot. Only you can talk to it. Nobody else can see your messages or access your data.
STEP 3

Deploy on Railway 5 min

Railway is a cloud hosting platform. It runs your AI 24/7 so you don't have to leave your laptop on. This is the "server" bit.

  1. Go to railway.com and sign up (use GitHub login, it's easiest)
  2. Click this one-click deploy link: Deploy OpenClaw on Railway
  3. It'll ask you to set some variables:
    SETUP_PASSWORD = (pick something, you'll need it once)
    PORT = 8080
  4. Click Deploy and wait ~2 minutes
  5. Once deployed, go to your service ‚Üí Settings ‚Üí Networking ‚Üí Enable Public Networking on port 8080
  6. Also go to Volumes ‚Üí Add a volume mounted at /data (this saves your stuff between restarts)
  7. Find your public URL in Settings ‚Üí Domains (something like your-thing.up.railway.app)
STEP 4

Run the Setup Wizard 3 min

Open your Railway URL with /setup at the end:

https://your-thing.up.railway.app/setup

Enter your SETUP_PASSWORD, then:

  • Paste your Anthropic API key from Step 1
  • Paste your Telegram bot token from Step 2
  • Select Claude Sonnet 4 as your model
  • Click Run Setup

That's it. That's the whole setup.

PRO TIPS

Things I Wish I Knew on Day 1

  • Voice notes work. Just send it a voice note on Telegram. It transcribes and responds. This is the killer feature - you can talk to your AI while driving, cooking, walking.
  • Give it a personality. Create a file called SOUL.md in the workspace. Tell it who it is, how to talk, what you like. Mine is called Atlas and it calls me on my bullshit.
  • Tell it about yourself. Create USER.md. Your timezone, your job, your preferences. The more it knows, the more useful it gets.
  • It has a web UI. Go to your-url.up.railway.app/openclaw for the control panel.
  • It remembers things. But only if you tell it to, or set up memory files. Ask it to "remember that I prefer X" and it'll note it down.
  • Start small. Don't try to do everything I'm doing on day 1. Just use it as a smart Telegram buddy for a week. Then start connecting things.
