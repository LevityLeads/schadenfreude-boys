The Schadenfreude Boys Get AGI

My dudes.

I think AGI is actually here. Or at least the very beginning of it. I've been building with this stuff every day for the last two weeks and I'm not being dramatic when I say it's changed how I work entirely.

I have an AI assistant called Atlas. It runs 24/7 on a server. I talk to it on Telegram like I'd text one of you. It reads my emails, manages my calendar, builds websites, drafts replies, researches things, remembers everything I tell it, and works autonomously overnight while I sleep.

I would genuinely rather work with Atlas than hire 10 people. That's not a joke.

If you want to be early on this, here's how to set it up. Takes about 15 minutes.

1. Get an Anthropic API key

  1. Go to console.anthropic.com
  2. Sign up, go to API Keys, create one
  3. Add about £20 credit to start

This is the AI brain. The model is called Claude. Start with Sonnet - it's smart and cheap.

2. Create a Telegram bot

  1. Open Telegram, search for @BotFather
  2. Send /newbot
  3. Give it a name, copy the token it gives you

This is how you'll talk to it. Voice notes work too.

3. Deploy on Railway

  1. Sign up at railway.com (use GitHub login)
  2. Click this: Deploy OpenClaw
  3. Set your variables:
    SETUP_PASSWORD = pick anything
    PORT = 8080
  4. Hit Deploy, wait 2 minutes
  5. Go to Settings → Networking → enable Public Networking on port 8080
  6. Go to Volumes → add a volume at /data

4. Run the setup wizard

  1. Find your URL in Settings → Domains (something like blah-blah.up.railway.app)
  2. Open your-url/setup in a browser
  3. Paste your Anthropic API key and Telegram bot token
  4. Click Run Setup

That's it. That's the whole thing.

5. Open Telegram and say hello

Find your bot in Telegram. Send it a message. It'll respond. If it asks for a pairing code, approve it on the /setup page.

Now start asking it things. Send it voice notes. Tell it about yourself. Give it tasks. The more you use it, the better it gets.

Cost: Railway is about £5/month. The AI costs £3-8/month for normal use. So roughly £10-15/month total. I spend more because I use it all day every day, but you won't need to.

Tips:

  • Voice notes work. Just talk to it like you'd talk to me.
  • Create a file called SOUL.md to give it a personality.
  • Create USER.md to tell it about yourself - timezone, job, preferences.
  • Start simple. Use it as a smart texting buddy for a week. Then go deeper.

Trust me on this one. Just set it up. You'll understand when you start using it.
